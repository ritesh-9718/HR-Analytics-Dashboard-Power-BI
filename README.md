# HR-Analytics-Dashboard-Power-BI
## Project Overview
This project presents an HR Analytics Dashboard built using Power BI to analyze employee data and identify key factors influencing attrition, workforce distribution, and job satisfaction.
The dashboard helps HR teams and management make data-driven decisions related to employee retention and workforce planning.

## Objectives

  - Analyze overall employee strength and attrition
  - Identify attrition patterns by department, age group, gender, and education
  - Evaluate job satisfaction levels across different job roles
  - Provide an interactive HR dashboard for quick decision-making

## Tools & Technologies Used

  -  Power BI Desktop
  -  Power Query – data cleaning & transformation
  -  DAX – calculated measures and KPIs
  -  Excel / CSV – data source

## Data Cleaning & Transformation

  -  Removed duplicate records
  -  Standardized categorical fields (Department, Education, Job Role)
  -  Created age bands for analysis
  -  Built DAX measures for:
      - Attrition Percentage
      - Active Employees
        
## Dashboard KPIs

  -  Overall Employees: 1,470
  -  Total Attrition: 237
  -  Attrition Rate: 16.12%
  -  Active Employees: 1,233

## Dashboard Features

  -  Department-wise attrition analysis
  -  Employee distribution by age group and gender
  -  Job satisfaction rating by job role
  -  Education-wise attrition insights
  -  Attrition rate comparison across age groups and gender
  -  Interactive slicers for education level

## Key Insights

  -  Highest attrition observed in the 25–34 age group
  -  Sales and R&D departments contribute the most to attrition
  -  Employees with Life Sciences and Medical backgrounds show higher attrition
  -  Job satisfaction varies significantly across roles
  -  Younger employees show relatively higher attrition trends
